## 📌 매월 고정 리마인더 (자동 체크)

오늘 날짜를 기준으로 아래 조건에 해당하면 **최상단(🔴 위)에 별도 블록으로 표시**해줘.

### 1. SR안전처 계산서 발행 (매월 25일 루틴)
- **기본 규칙**: 매월 25일 세금계산서 발행해서 올려야 함 (내부 업무)
- **주말 회피 규칙**: 25일이 주말이면 직전 금요일로 앞당김
  - 25일 = 평일 → 당일 25일
  - 25일 = 토요일 → 24일(금)
  - 25일 = 일요일 + 24일 = 토요일 → 23일(금)
- **알림 대상 날짜 판단**:
  - 오늘이 해당 "조정된 발행일"이면 🚨 **"오늘 SR안전처 계산서 발행"** 표시
  - 오늘이 조정된 발행일 D-1이면 📅 **"내일 SR안전처 계산서 발행"** 표시
- **공휴일 예외**: 12월 25일(크리스마스)처럼 평일이어도 공휴일이면 직전 영업일로 앞당겨 안내

판단 예시 (2026~2027):
| 월 | 원래 25일 | 실제 발행일 |
|---|---|---|
| 2026-04 | 토 | 04-24(금) |
| 2026-05 | 월 | 05-25(월) |
| 2026-07 | 토 | 07-24(금) |
| 2026-10 | 일 (24일도 토) | 10-23(금) |
| 2026-12 | 금·크리스마스 | 12-24(목) 권장 |
