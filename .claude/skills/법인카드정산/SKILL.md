# 법인카드 기안품의서 작성

## 참조 파일
- **양식**: `..private/양식/법인카드_기안품의서_2026.xls`
- **예시**: `..private/양식/법인카드_기안품의서_예시.png`
- **위임전결규정**: `..private/양식/위임전결규정_요약.md` (결재라인·송금일 기준)

## 계정 분류 기준 (자동 분류에 사용)

| 계정 | 해당 항목 |
|------|---------|
| 복리후생비 | 점심 식대, 사내 회식, 사내 간식 |
| 여비교통비 | 외근·출장 시 식음료비, 숙박비, 교통비, 법인차량 주차비 |
| 차량유지비 | 법인차량 주유비, 법인차량 정비비 |
| 통신비 | 등기 발송 비용, 사내 통신비 |
| 소모품비 | 업무상 필요한 비품 구매 |
| 지급수수료 | 인증·증명서 발급 비용, 각종 수수료 |
| 접대비 | 거래처 접대 비용 |

## 실행 순서

### Step 1: 카드 사용내역 받기
사용자에게 아래 중 하나를 요청:
- 결제예정내역 엑셀/텍스트 붙여넣기
- 또는 직접 입력 (승인일, 가맹점명, 금액)

최소 필요 정보: **승인일, 가맹점명, 금액**

### Step 2: 자동 계정 분류
각 사용건을 위 계정 분류 기준에 따라 자동 분류한다.
- 가맹점명으로 추론 (예: 주유소 → 차량유지비, 식당 → 복리후생비)
- 확실하지 않은 건은 사용자에게 확인

### Step 3: 사용 목적 자동 생성
계정에 맞는 사용 목적을 작성한다.
- 복리후생비 → "사내 간식 구매", "사내 점심 식대" 등
- 여비교통비 → "출장 점심 식대", "법인 차량 주차비" 등
- 차량유지비 → "법인 [차종] 주유대" 등
- 사용자가 보정 가능

### Step 4: 기안품의서 테이블 출력
아래 형식으로 정리해서 보여준다:

```
■ 법인카드 지출내역 (YYYY년 MM월)

| 사용일자 | 사용처 | 계정 | 금액 | 사용 목적 | 사용자 |
|---------|--------|------|------|---------|--------|
| YYYYMMDD | 가맹점 | 계정명 | 금액 | 상세내용 | 이름 |

카드사용계: ₩X,XXX,XXX

■ 합계 (계정별)
| 계정 | 금액 |
|------|------|
| 여비교통비 | ₩XX,XXX |
| 차량유지비 | ₩XX,XXX |
| ... | ... |
| 카드사용계 | ₩X,XXX,XXX |
```

### Step 5: 사용자 확인
- 분류가 맞는지 확인
- 수정사항 반영
- 확정되면 "엑셀에 옮기면 됩니다" 안내

### Step 6: 완료
```
✅ 법인카드 기안품의서 작성 완료
📊 총 N건 / ₩X,XXX,XXX
📂 양식: ..private/양식/법인카드_기안품의서_2026.xls
```

## 주의사항
- 사용자 이름 기본값: 사용자 본인 (대리)
- 법인차량 차종은 첫 정산 시 확인 후 기억해둘 것
- 계정 분류 애매한 건은 반드시 사용자 확인
