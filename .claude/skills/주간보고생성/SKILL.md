# 주간보고 새 시트 자동 생성

## 개요
매주 월요일에 실행 → 이번 주 금요일 주간보고 시트를 미리 생성한다.
최신 시트를 복사하여 새 시트를 만들고, 필요한 칸만 비운다.

## 파일 경로
- **엑셀 파일**: `주간보고/주간보고_영업부.xlsx`
- **생성 스크립트**: `주간보고/주간보고_생성.py`

## 실행 방법

### Step 0: 문서점검 선행 (필수)
주간보고 시트 생성 **전에** `/문서점검` 스킬을 먼저 실행한다.
- 목적: TODO·파이프라인·고객 정보 불일치를 잡아둔 뒤 보고 시트를 채워야 일관된 숫자가 들어감.
- 불일치 0건 → 바로 Step 1 진행
- 불일치 있으면 → 사용자에게 결과 보고 후 "수정하고 진행할까요, 일단 시트만 생성할까요?" 확인
- 사용자가 "그냥 진행"이면 Step 1로, "수정"이면 문서점검 Step 4 수정 흐름 → 완료 후 Step 1

### 1. 날짜 지정 시
사용자가 날짜를 지정하면 해당 날짜로 생성:
```bash
python3 "주간보고/주간보고_생성.py" {YYYY-MM-DD}
```

### 2. 날짜 미지정 시
이번 주 금요일 날짜로 자동 생성:
```bash
python3 "주간보고/주간보고_생성.py"
```

## 생성 규칙
1. 최신 시트(첫 번째)를 복사하여 새 시트 생성
2. 시트명 = `YY.MM.DD` (작성일)
3. C2(작성일) 변경 → C5, H5 날짜 범위는 수식으로 자동 계산
4. **유지**: C열(전주 계획), A열(부서), B열(구분)
5. **비우기**: D열(계획진행율), E열(실적), F열(진행률), G열(담당자), H열(차주 계획), I열(Due-Date), J열(담당자)
6. D~J 전체 열에 C열과 동일한 셀 병합 적용
7. 틀 고정: A7 (6행까지 고정)

## 실행 후 안내
생성 완료 후 사용자에게 알려줄 것:
- 생성된 시트명
- 직접 작성해야 할 항목: 계획진행율(D), 실적(E), 진행률(F), 차주계획(H), Due-Date(I), 담당자(J)
- "엑셀 열어서 확인해보세요"

## 주의사항
- 엑셀 파일이 열려 있으면 저장 실패 → "엑셀 닫고 다시 시도해주세요" 안내
- 이미 같은 날짜 시트가 있으면 중복 생성하지 않음
