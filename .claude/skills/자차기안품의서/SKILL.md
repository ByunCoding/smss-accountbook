# 자차 외근경비 기안품의서 작성

## 참조 파일
- **기안 양식·계산 공식**: `c:\Users\user\Desktop\자차경비.txt`

## 사용자 차량 정보 (고정값)
- **차종**: 23년형 KGM 토레스
- **공인연비**: 10.2 km/L
- **입금계좌**: 국민은행 72720200012928 변상민

## 계산 공식
```
거리(KM) / 10.2 × 오피넷 경기 평균 휘발유가 × 1.2(감가상각비) = 금액
```
- 소수점 이하 올림
- 휘발유가: **오피넷 경기 기준** 해당 날짜별 조회

## 결재선
- 기안품의서: 김우곤 부장님 + 대표님
- 지출결의서: 김우곤 부장님께만

## 실행 순서

### Step 1: 외근 내역 수집
사용자에게 아래 정보를 요청:
- **정산 기간** (몇 월분?)
- **외근 내역**: 날짜, 장소, 용도, 왕복 거리(KM)
- **하이패스 PDF 경로** (있으면 — 통행료 자동 분석용)

최소 필요 정보: **날짜, 장소, 거리(KM)**

### Step 2: 오피넷 휘발유가 조회
각 외근 날짜별로 **오피넷 경기 기준 해당일 평균 휘발유가**를 조회한다.
- 날짜별로 가격이 다르므로 반드시 **각 날짜별 개별 조회** 필요
- WebSearch 또는 WebFetch로 오피넷(opinet.co.kr) 경기 지역 조회
- 과거 날짜 조회 불가 시 사용자에게 직접 확인 요청

### Step 3: 하이패스 PDF 분석 (통행료)
사용자가 하이패스 PDF를 제공한 경우:
1. PDF를 Read 도구로 읽어 전체 통행 내역 추출
2. 각 건의 **거래일자, 출구 영업소명, 이용금액** 파싱
3. **외근 날짜별로 그룹핑** — 해당일의 통행료만 묶음
4. 날짜별 통행료 행 데이터 생성:
   - **합계금액**: 해당일 통행료 합산 + "원" 붙임
   - **비고**: 출구영업소(금액)+출구영업소(금액) 형식 (= 합계 없이)
   - 예시: `서수지(1,000)+금토2(1,100)+기흥(1,680)`

### Step 4: 금액 계산
각 건별로 계산:
```
거리(KM) / 10.2 × 휘발유가 × 1.2 = 금액 (소수점 올림)
```

### Step 4: 기안품의서 전체 출력
**마크다운 테이블(칸 구분)** 형식으로 출력한다. 사용자가 각 칸의 내용을 보고 엑셀 셀에 직접 입력할 수 있게 한다.
