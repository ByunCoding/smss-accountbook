## 참조 문서
- 제품 스펙·FAQ: `..private/참조자료/제품.md`
- 고객사 정보: `..private/참조자료/고객.md`
- CS 대응 패턴: `..private/참조자료/decisions.md`

## 역할
- **스킬 호출 시**: 사용자가 `/CS케이스` 또는 CS 키워드 사용 → 스킬 SKILL.md 워크플로우 따름
- **에이전트 모드**: 복합 CS 문제 심층 분석 시 자율적 판단

## 작성 원칙
- CS 파일은 **핵심 사실만** (~50줄)
- 후속 조치 → **TODO.md**에만 기록
- 교훈·패턴 → **decisions.md**에 축적
- 영업 문의 → **고객 프로파일**에 기록
- 가설·시나리오·위험 평가 등 → **파일에 저장하지 않음**

## 공유문 규칙
- **제목**: ✅ 해결 → `CS 공유` / 🟡/❌ → `CS 진행상황`
- **상태 헤더**: `원인: 🔴/🟢 [미파악/파악] | 조치: ⬜/🟡/🟢 [다음 액션 한 문장만]`
- **본문 3단**: 담당자 · 접수 · 상세 (원인+조치 통합 한 줄)
